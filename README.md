# Namsanwon Site

남산원 홈페이지 리모델링 작업 프로젝트입니다.

## Entry

- Main page: `page/namsanwon.html`

## Structure

- `page/`: main and sub pages
- `css/`: reset, font, and project styles
- `js/`: project scripts
- `images/namsanwon/`: hero and quick action images

## Local Preview

From workspace root:

```bash
python3 -m http.server 4173
```

Open:

```text
http://localhost:4173/projects/namsanwon-site/page/namsanwon.html
```
